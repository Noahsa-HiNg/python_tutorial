Lý Thuyết Mảng Trong Python

1. Mảng là gì?

Mảng là cấu trúc dữ liệu dùng để lưu nhiều giá trị trong cùng một biến. Trong Python, mảng thường được sử dụng dưới dạng List. 

a = [1, 3, 5, 7, 9]

Mảng trên có 5 phần tử: 1, 3, 5, 7, 9

2. Chỉ số của mảng

Trong Python, chỉ số bắt đầu từ 0.

Chỉ số Giá trị
a[0] 10
a[1] 20
a[2] 30
a[3] 40 
a = [10,20,30,40]

print(a[0])

3. Khai báo mảng

Khai báo trực tiếp:

a = [1,2,3,4,5]

Tạo mảng rỗng:

a = []

Tạo mảng gồm nhiều số giống nhau:

a = [0] * 5

4. Nhập mảng từ bàn phím

n = int(input())
a = list(map(int,input().split()))

Ví dụ input:

5
1 2 3 4 5

5. Duyệt mảng

Duyệt theo chỉ số:

for i in range(len(a)):
    print(a[i])

Duyệt trực tiếp:

for x in a:
    print(x)

6. Các thao tác cơ bản

Thêm phần tử:

a.append(10)

Xóa phần tử:

a.remove(3)

Độ dài mảng:

len(a)

7. Bài toán thường gặp với mảng

Tính tổng mảng:

s = 0
for x in a:
    s += x

print(s)

Tìm số lớn nhất:

max_value = a[0]

for x in a:
    if x > max_value:
        max_value = x

print(max_value)

Đếm số chẵn:

count = 0

for x in a:
    if x % 2 == 0:
        count += 1

print(count)

8. Sắp xếp mảng

Sắp xếp tăng dần:

a.sort()

Sắp xếp giảm dần:

a.sort(reverse=True)

9. Hàm có sẵn

max(a)
min(a)
sum(a)

10. Tóm tắt

  • Mảng lưu nhiều dữ liệu trong một biến
  • Chỉ số bắt đầu từ 0
  • Duyệt mảng bằng vòng lặp for
  • Các bài cơ bản: tổng, max, min, đếm
